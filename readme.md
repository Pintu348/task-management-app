# Task Management APP

This Task Management App is built using React.js and Tailwind CSS, providing users with a simple yet efficient tool for organizing tasks and subtasks. Leveraging the capabilities of localStorage, the app offers persistence across browser sessions, ensuring that users can access their tasks anytime, anywhere without the need for server-side storage.Features:

## Feature

**Adding Tasks:** Users can easily add tasks to the app by entering the task title and description. Each task can have multiple subtasks associated with it.

**Adding Subtasks:** For better organization, users can create subtasks within each task, allowing for granular management and tracking of progress.

**Removing Tasks and Subtasks:** Users have the flexibility to remove both tasks and their associated subtasks, providing control over their task list.

**LocalStorage Persistence:** The app utilizes the browser's localStorage feature to persist tasks and subtasks across sessions, ensuring data persistence and easy access.

## Technologies Used:

**React.js:** The app's frontend is developed using React.js hooks like useState, useRef,portals etc, providing a responsive and dynamic user interface.

**Tailwind CSS:** Tailwind CSS is employed for styling, offering a utility-first approach for rapid development and customization.

**LocalStorage:** Browser's localStorage feature is utilized for data persistence, allowing users to store and access their task data locally.
